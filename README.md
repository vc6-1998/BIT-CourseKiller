# BIT-CourseKiller
针对北京理工大学选课网站的抢课程序，支持公选课和体育课。
## 环境配置及运行
* 先配置好Python基础解释器，然后在项目目录下运行以下命令安装需要的第三方库（requests, beautifulsoup4, pycryptodome）
```shell
pip install -r requirement.txt
```
* 编辑`course.txt`，里面换成你想要的课，一行放一门课，目前支持关键字搜索
* 运行主程序：
```shell
python main.py
```
## 注意事项
* 一定要在控制台中运行命令来启动程序，如果你用的 Pycharm 之类的IDE请不要直接点上面的“运行”按钮，否则密码会无法输入
* 正常情况下密码输入的内容是隐藏的，控制台中不会显示，防止隐私泄露
* 当前课程搜索是按关键字搜索的，和在网站右上角搜索方式是一样的，且会筛选掉冲突的课程
* 程序运行韧性非常强，不会因为断网而退出，相反运行的时候压根进不去选课网站
